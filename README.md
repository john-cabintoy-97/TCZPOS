# TCZ-POS 🛒

A lightweight, high-performance **Point of Sale (POS) ** built using a cross-platform desktop and mobile hybrid architecture. 

TCZ-POS combines the native power of **.NET MAUI** with the rapid UI flexibility of **Blazor Hybrid** and **Tailwind CSS**.

---

## 🚀 Features

*   **Cross-Platform Deployment:** Runs natively on Windows and Android (optimized for POS terminals and tablets) from a single codebase.
*   **Integrated Queuing System (QPOS):** Live queuing, ticket tracking, and state management for customer order flows.
*   **Offline-First Architecture:** Leverages client-side local storage and local databases to keep sales moving without an internet connection.
*   **Hardware Integration:** Built-in support for hardware barcode scanners, QR code generation, and camera-based scanning.
*   **Modular Web UI:** Utilizes headless UI components styled with Tailwind CSS for zero-overhead, highly accessible interfaces.

---

## 🛠️ Tech Stack

*   **Application Framework:** .NET MAUI (Multi-platform App UI)
*   **UI Architecture:** Blazor Hybrid (Razor Components running natively)
*   **Styling & Components:** Tailwind CSS & Ark UI (Headless components)
*   **Package Management:** Nuget (.NET) & NPM/Node.js (Frontend JS Interop & Utilities)
